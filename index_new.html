Production Tools

AI Generators

These tools were built the same way I approach a camera setup — every parameter intentional, nothing left to chance. If you've never used AI for production before, or you're skeptical of what it can actually do, this is the right place to start.

You don't need to know anything technical. Describe what you want to see — a scene, a character, a lighting setup — and the generator handles the rest. The style, the camera spec, the lighting logic: it's already built in. You're just providing the subject.

The generators run on a tiered engine stack — from FLUX 1.1 Pro Ultra (the top of the public stack) down to a free no-key fallback. The engine used is shown on each card. Add your own API keys in ⚙️ Settings to unlock higher quality tiers — keys stay on your device only.

TIER 1 — fal.ai · FLUX 1.1 Pro Ultra BEST QUALITY · ~$0.06/image

The absolute top of the public stack right now. Get a free key at fal.ai/dashboard/keys → Create Key. Free credits on signup.

TIER 2 — Together.ai · FLUX.1-dev HIGH QUALITY · ~$0.013/image

Same FLUX.1-dev model but on dedicated fast GPU. Get a key at api.together.ai → API Keys. Free $1 credit on signup.

TIER 3 — HuggingFace · FLUX.1-dev FREE · Slower

Free with an HF account. Get a token at huggingface.co/settings/tokens → New Token → Read. Then accept the model license at FLUX.1-dev page.

TIER 4 — Google Gemini · Image Analysis FREE · Sketch uploads

Enables sketch/photo analysis when uploading reference images. Free key at aistudio.google.com → Get API Key.

All keys stored locally on your device only. Never sent to any server other than the respective AI provider.
