CINEMATIC ENGINE

Storyboard → Veo 3 Spec

01

[ HOW IT WORKS ]

Upload a rough storyboard frame. The engine analyzes composition, lighting, and subject placement, then generates a complete camera move specification optimized for Veo 3 video generation.

[ OUTPUT INCLUDES ]

  • Scene analysis with lighting breakdown
  • Equipment specs (camera, lens, rig)
  • Mark A → Vector → Mark B coordinates
  • Veo 3 prompt with NO_MORPH token
  • 5 clarifying questions for refinement

IMAGE GENERATOR

Scene → Cinematic Still

02

[ HOW IT WORKS ]

Describe a scene in plain language. The generator wraps your description in VEFILM's cinematic prompt architecture: Arri Alexa, Zeiss Master Prime, Kodak Vision3 500T, period-accurate textures.

[ BUILT-IN PARAMETERS ]

  • Kodachrome 64 film emulation
  • Anamorphic 2.39:1 framing
  • Deep shadow blacks, warm practicals
  • No digital smoothing or AI tells
  • Period-accurate production design

VECTOR-DP

Location → Spatial Analysis

03

[ HOW IT WORKS ]

Upload a location photo. VECTOR-DP performs perspective correction, estimates focal length, extracts an orthographic floor plan, and calculates optimal lighting vectors for your shoot.

[ OUTPUT INCLUDES ]

  • Vanishing point analysis
  • Estimated focal length from perspective
  • Top-down floor plan with coordinates
  • Light stand placement vectors
  • Camera plate suggestions

04

[ HOW IT WORKS ]

Describe a character in text. The engine generates a comprehensive character design sheet with wardrobe breakdown, expression variations, and consistency parameters for AI generation.

[ OUTPUT INCLUDES ]

  • Physical baseline (age, build, features)
  • Wardrobe breakdown by layer
  • Expression sheet (4 emotional states)
  • Consistency lock parameters
  • MidJourney/Veo prompt fragments
