## HOW TO USE THIS DOCUMENT

Every prompt ends with a **PARAMETERS BLOCK** — always paste it exactly as written.
The **BASE PARAMETERS** lock in realism, suppress MidJourney's stylization instincts, and enforce cinematic framing.

**Base Parameters (always include):**
```
--ar 16:9 --v 6.1 --style raw --s 80 --q 2
```

**For extreme close-ups / portrait shots:**
```
--ar 4:5 --v 6.1 --style raw --s 80 --q 2
```

**For 2.39:1 cinemascope (ultra-wide cinematic):**
```
--ar 239:100 --v 6.1 --style raw --s 80 --q 2
```

# SECTION 2: THE INN — LOCATION PROMPTS

## Location Baseline (LOCK)
**CONFIRMED FROM REFERENCE IMAGES — DO NOT DEVIATE**
- Scotland, circa 1967
- Two-story dark grey/near-black render or stone — structure blends into the dark sky
- Simple rectangular massing — roadside country pub/inn, not a grand hotel
- Slate pitched roof, two prominent chimneys
- Minimal windows — one or two ground floor windows with very dim amber interior glow
- Small hanging lantern or pub sign near entrance — single amber-red point of light
- Set directly on a narrow winding country lane — wet reflective asphalt, NO cobblestones
- Bare silhouetted trees flanking both sides of the property
- Dense fog consuming the building edges — you see the light before you see the structure
- Sky: near-total black, no moonlight, no stars
- Interior palette: dark amber, near-black shadow, warm practical lighting only
- Materials: dark plaster walls, dark wood trim, worn wood floors, stone in some areas
- No modern elements — no fluorescent light, no visible electronics
